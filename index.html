Git Mastery

Git Mastery is designed to provide students with developing a strong intuition for how the Git version control system works through practical problem sets designed to run entirely on their local machines.

What Git Mastery offers

Are you an instructor trying to integrate Git Mastery into your Git curriculum?

Git Mastery is very simple to integrate into any existing Git curriculum. You can pick and choose the problem sets that your students should complete and Git Mastery provides the necessary progress monitoring tooling for it.

Educators Resource

Are you a student who wants to bring their Git understanding to the next level?

Git Master allows you to practice Git through problem sets designed to replicate real-world scenarios. Get directed feedback for every submission and monitor your progress easily.
